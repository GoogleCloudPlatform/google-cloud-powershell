What is it?

Cloud Tools for PowerShell is a set of PowerShell cmdlets for accessing and manipulating Google Cloud Platform resources. With them you will be able to automate tasks by writing programs in PowerShell. And ultimately get more down on the Google Cloud Platform.

Cloud Tools for PowerShell uses the same configuration and credentials used by the Google Cloud SDK (gcloud). Once you authenticate (gcloud auth login) you will be able to use PowerShell cmdlets.

FAQ

What is the relationship between the Cloud Tools for PowerShell and the gcloud command-line tool?

Both the gcloud command-line tool and Cloud Tools for PowerShell cmdlets allow you to manage your cloud resources. However, the Cloud Tools for PowerShell returns strongly-typed objects rather than plain text. So cmdlet results can be latter transformed within Windows PowerShell. With Cloud Tools for PowerShell you can benefit from things like tab-completion, type checking, and integrated help.
